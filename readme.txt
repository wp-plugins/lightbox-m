== Description ==

This plugin includes the mediaboxAdvanced v1.1.7 javascript written by John Einselen and got transformed into a Wordpress Plugin by CheonNii.
From settings you can choose box style add the lightbox effect to your post's images and media.
Use the title attribute if you want to show a caption.
[Plugin Homepage](http://cheon.info/707/ "Plugin Homepage")

== Installation ==

**Upgrade**

1. De-activate the plugin.
1. Delete the old version `lightbox-cheon.info` directory

**Install**

1. Unzip the `lightbox-cheon.info.zip` file.
1. Upload "lightbox-cheon.info" folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress, go to Setting->Lightbox M
3. Add a rel="lightbox" attribute to any link tag to activate the lightbox or rel="lightbox[roadtrip]" for image set
